# Stacking Paper: Building on ML Research

Welcome to **Stacking Paper(s)**! This is my personal playground for breaking down the latest machine learning research papers, implementing them from scratch, and bridging the gap between intertwined topics; all while documenting my findings along the way.

Rather than just posting finished code, I use this space to write articles detailing my implementation struggles, architectural decisions, and the "gotchas" that aren't mentioned in the original papers.
